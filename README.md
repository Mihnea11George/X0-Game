# X0 Game 🎮

Un joc clasic de X și 0, făcut cu HTML, CSS și JavaScript curat (vanilla). L-am creat ca un mic proiect pentru a exersa lucrul cu grid-uri în CSS și manipularea DOM-ului în JS.

## Tehnologii folosite
* **HTML5** pentru structură
* **CSS3** (CSS Grid) pentru aspect și tabla de joc
* **JavaScript** pentru logica de joc (rânduri, detectare victorie/egalitate)
